# Project_seminar.Volkov_Dmitry
## СИСТЕМА МОНИТОРИНГА СОСТОЯНИЯ ВОДИТЕЛЯ

Этот проект представляет собой работоспособное ядро, реализующее систему мониторинга усталости на основе адаптивного порогового подхода в реальном времени на стандартном CPU в рамках MVP.

## Созданный прототип:
- Работает в реальном времени на стандартном CPU.
- Адаптируется под водителя за 30 секунд.
- Обладает прозрачной логикой, основанной на статистически обоснованных коэффициентах.
- Прошел всестороннее тестирование и доказал свою работоспособность.
- Позволяет сохранять результаты обучения в базе данных SQLite.
- Позволяет экспортировать результаты в файлы.
- Содержит ячейки для калибровки по фотофайлу и анализировать видеофайл.
