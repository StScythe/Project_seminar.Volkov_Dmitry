# Project_seminar.Volkov_Dmitry
## СИСТЕМА МОНИТОРИНГА СОСТОЯНИЯ ВОДИТЕЛЯ

Этот проект представляет собой работоспособное ядро, реализующее систему мониторинга усталости на основе адаптивного порогового подхода в реальном времени на стандартном CPU в рамках MVP.

## Созданный прототип:
а)	Работает в реальном времени на стандартном CPU.
б)	Адаптируется под водителя за 30 секунд.
в)	Обладает прозрачной логикой, основанной на статистически обоснованных коэффициентах.
г)	Прошел всестороннее тестирование и доказал свою работоспособность.
д)  Позволяет сохранять результаты обучения в базе данных SQLite.
е)  Позволяет экспортировать результаты в файлы.
ж)  Содержит ячейки для калибровки по фотофайлу и анализировать видеофайл.
